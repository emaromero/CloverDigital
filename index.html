Looki Bar

Nuestro proyecto más reciente es Looki Bar, una interfaz moderna desarrollada en React JS, codificada en Visual Studio Code y desplegada en Vercel, para un negocio de delivery y take away. A pedido de Matías, el CEO, implementamos un sistema de pedidos online que se envían directamente a WhatsApp. El menú es gestionado por el cliente desde una hoja de cálculo de Google, permitiéndoles modificar, activar o desactivar productos de forma dinámica. Esta solución combina tecnología avanzada con usabilidad, optimizando la operación del negocio.
